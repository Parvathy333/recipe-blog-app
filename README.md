Recipe Blog App
A simple and clean recipe blog where users can browse recipes and add their own recipe ideas. 
Live Demo: https://parvathy333.github.io/recipe-blog-app/

Features
View a list of recipes
Open any recipe to see ingredients and steps
Add your own recipe ideas
Responsive and lightweight UI

Tech Used
HTML
CSS
JavaScript
